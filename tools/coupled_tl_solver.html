📐 Formula Reference

Even / Odd Mode Inductance

Le = Ls + Lm
Lo = Ls − Lm

Even / Odd Mode Capacitance

Ce = Cs − Cm
Co = Cs + Cm

Characteristic Impedance

Z₀e = √(Le / Ce)
Z₀o = √(Lo / Co)

Propagation Velocity

ve = 1 / √(Le · Ce)
vo = 1 / √(Lo · Co)

Propagation Delay

te = ℓ / ve
to = ℓ / vo
Δt = to − te (delay skew)

Reflection Coefficients

ΓL = (ZL − Z₀) / (ZL + Z₀)
ΓS = (ZS − Z₀) / (ZS + Z₀)

Initial Incident Wave

V⁺ = VS · Z₀ / (Z₀ + ZS)

Crosstalk (NEXT / FEXT)

NEXT ∝ (Lm/Ls + Cm/Cs) / 4
FEXT ∝ (Lm/Ls − Cm/Cs) · ℓ / (2·T)
